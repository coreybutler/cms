Headers

Use # next to a header. The more # signs, the smaller the header.

# Header 1 (h1)
## Header 2 (h2)
### Header 3 (h3)

...

Text

Markdown supports bold and italic text.

**text**
= text

__text__
= text

_text_
= text

*text*
= text

You can also create links for specific text. The format is:

[title](http://mylink.com)

Linking to google might look like:

Go to [Google](http://www.google.com)

Go to Google

Pictures

Adding a picture is just like adding a link, except it has an exclamation point first:

![The Google Logo](https://www.google.com/images/srpr/logo11w.png)

Lists

Creating lists simply requires a dash where you want the bullet point.

- Item A
- Item B
- Item C

Renders:

  • Item A
  • Item B
  • Item C

    • Creating numbered lists simply requires typing the number and a period.

    1. Item A
    2. Item B
    3. Item C

    Renders:

    1. Item A
    2. Item B
    3. Item C

    Horizontal Line

    A horizontal line is created using three dashes:

    ---
